# 破解

将/jetbra文件夹复制到电脑某个路径，且路径不能含有空格和中文

  • Mac：在终端中打开这个文件夹，进入/jetbra/scripts执行install.sh 脚本
  • Win：点击进入 /jetbra 补丁目录，再点击进入 /scripts 文件夹，双击执行 install-current-user.vbs 脚本：

# 激活
